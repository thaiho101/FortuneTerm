# Market Cost – Budget Tracker

## 🚀 Overview
A full-stack budgeting application built to track personal spending and prevent overspending through structured expense management and data visualization.

## 🔥 Key Features
- Budget creation and expense tracking
- Categorized transaction logging
- Interactive dashboards for spending insights
- Simple and clean UI for daily use

## 📊 Impact
- Actively used for managing personal finances
- Improved visibility into spending patterns and budgeting habits

## 🛠 Tech Stack
- PHP
- MySQL
- JavaScript
- HTML/CSS
